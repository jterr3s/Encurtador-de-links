# Encurtador-de-links
Projeto de um encurtador de links

## 📌 Sobre o Projeto
  O projeto tem como objetivo receber um link qualquer, e devolver um novo link com 6 digitos que encaminhara o usuário para o mesmo site do endereço recebido
  
## 🛠️ Tecnologias Utilizadas
*Python
*Flask
*Flask-SQLAlchemy
